# Kos-Sanz — Web Aplikasi Pencari Kos Online

Web aplikasi manajemen kos multi-peran (Admin, Pemilik, Penghuni) 
dengan arsitektur serverless.

## Tech Stack
- React.js + Vite
- Firebase Firestore (database)
- Firebase Authentication
- Netlify (hosting)

## Fitur Utama
- Autentikasi berbasis peran (Admin / Pemilik / Penghuni)
- Manajemen kamar (CRUD) dengan foto & lokasi Google Maps
- Sistem booking real-time
- Pembayaran via QR code + verifikasi bukti oleh Admin
- Manajemen komplain penghuni
- Tampilan Dark Mode

## Demo
🔗 https://kos-sanz.netlify.app
